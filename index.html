PROJECTS

WORK

SCHOOL

During my co-op at New York Magazine, I developed a reusable quiz template component with customizeable scoring on their Node-based CMS for editors to use. The quiz types supported were multiple choice, scale, and a simple agree/disagree quiz. It was built with Handlebars.js, SCSS, and client-side JavaScript.

My first major project at New York Magazine was to build a bar and line chart visualization component. The idea is to allow editors to create these charts by just inputting data and selecting a few customizations. The article linked to the left used the base foundation of the component I developed, and it also added some changes to style and optimization. The component was built with Handlebars.js, SCSS, client and server-side JavaScript, and d3.js to spin up the visualizations.

Another project I worked on at New York Magazine was a 100 Jokes interactive feature. I developed the component that was used to add x-amount of items in the list (in this case 100 jokes), as well as the component that acted as a container for that.

Initial web application assignment for my Web Development class (CS4550 - Fall 2017): a simple microblog application. A registered user can post messages, follow other users, and like messages. Admin user feature was added for deleting any user or message. Built with Elixir + Phoenix for the overall app structure and backend (implementing websockets for live updates). DNS hosting on Namecheap and VPS hosting by Vultr.

Course project for my Web Development class (CS4550 - Fall 2017): MBTA Tool that lets users check for MBTA alerts and bus stops near them. Built with React.js for the front end framework, the Google Maps API for map rendering and geocoding, Elixir + Phoenix for the backend and implementing websockets/channels, and the MBTA API for querying through stops and alerts.

Final project for my Database Design class (CS3200 - Fall 2016): a New York City and Boston-based taco locator. You can choose your location/neighborhood and view the top taco places nearby, and if you're a logged in user, you can read/write reviews for each place. It was built with JavaScript, HTML/CSS, MySQL, and the Google Maps API.
